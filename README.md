# 機材予約システム

Vue3 と Element Plus を利用した簡易的な機材予約システムです。アクセスするとまずログイン画面が表示され、ログイン後に3Dプリンターやレーザー機の予約を行うことができます。データはブラウザ上に保存されるだけの簡単なサンプルです。

## 使い方
1. `index.html` をブラウザで開きます。
2. ユーザー名とパスワードを入力し「ログイン」を押します。
3. 機材と開始・終了日時を入力して「予約」を押すと、下の表に追加されます。

UI ライブラリとして Element Plus を CDN 経由で読み込んでいます。実運用する際は適切なバックエンド認証や予約管理を組み込んでください。
